jquery-modalMsg
===============

Plugin for simple configurable modal popup

## Get your modal in four simple steps:

1. Get the sources
2. Prepare your modal HTML structure
3. Style your modal as you wish
4. Attach the plugin!

For more info see the look at the examples.

## For development

1. Download git repo.
2. run `npm install`
3. run `bower install`
4. Do your work
5. Use grunt.js
